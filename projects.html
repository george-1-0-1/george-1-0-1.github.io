Project Report Code

Technical Stack

ROS 2 Python DDS Gazebo MATLAB/Simulink Machine Learning

  • Formation control algorithms
  • Real-time sensory data fusion

Presented at the International Conference on Electronics, AI and Computing (2025). This research contributes to the advancement of coordinated autonomous systems for real-world applications.

Key Features
